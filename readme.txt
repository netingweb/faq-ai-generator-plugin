Automatically generate FAQs for your WordPress content using artificial intelligence.

== Description ==

FAQ AI Generator is a WordPress plugin that uses OpenAI's artificial intelligence to automatically generate relevant FAQs for your content. The plugin is perfect for improving your site's SEO and providing immediate answers to your visitors' most common questions.

= Key Features =

* Automatic FAQ generation using GPT-4
* Schema.org integration for better SEO
* Full support for Advanced Custom Fields (ACF)
* Intuitive admin interface
* Multilingual support (English and Italian)
* Compatibility with posts, pages, and custom post types
* Integrated debug system
* Advanced HTML content management
* Support for nested and complex ACF fields

= Requirements =

* WordPress 5.0 or higher
* PHP 7.4 or higher
* Valid OpenAI API Key

== Installation ==

1. Upload the `faq-ai-generator` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Settings > FAQ AI Generator' to configure your OpenAI API Key
4. Start generating FAQs for your content!

== Frequently Asked Questions ==

= How does the plugin work? =

The plugin analyzes your page or article content and uses OpenAI's artificial intelligence to generate relevant FAQs. The FAQs are then automatically added to your content and structured according to schema.org for better SEO.

= Do I need an OpenAI API Key? =

Yes, a valid OpenAI API Key is required to use the plugin. You can get one by signing up at [OpenAI](https://platform.openai.com).

= Does the plugin support ACF? =

Yes, the plugin fully supports Advanced Custom Fields (ACF), including nested and complex fields. All ACF fields are analyzed and used to generate relevant FAQs.

= Can I customize the generated FAQs? =

Yes, you can modify, add, or remove FAQs directly from the plugin's admin interface.
